Default accounts:

- `superadmin` / `admin123`
- `admin` / `admin123`
- `advisor` / `admin123`

## Roles

- `super_admin`: full access, including user management and audit logs
- `admin`: can manage customers, vehicles, service types, repair orders, and assigned services, and can view audit logs
- `advisor`: can manage customers, vehicles, repair orders, and assigned services
